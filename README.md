# SCSS Task

This repository contains a simple SCSS project with the following files:

- `main.html` — sample HTML page
- `style.scss` — SCSS source file
- `style.css` — compiled CSS output
- `style.css.map` — source map for the compiled CSS

## Usage

Open `main.html` in your browser to view the sample page.
